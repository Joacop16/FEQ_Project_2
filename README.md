# FEQ_Project_2
This repository contains project 2 for the first-experience-in-quantum-research program. The objective is study the alternative methods such as Rungekutta and Taylor expansion for open system dynamics with sparse matrices.

Note: The code is written in Julia, if you are not familiar take a look on https://cheatsheet.juliadocs.org/ and https://cheatsheets.quantecon.org/ .
